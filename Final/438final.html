Overview and Usage

    In my game, I replicated a two dimensional basketball game which awards the user points at 10 point increments, and once the user reaches 40 points the player wins and can choose to play again. In order to run the code you must click the link below. Mooshball To play the game, you can click on the screen, and the ball will follow the mouse. In order to get points, you must aim the ball into the hoop or bounce off the wall.

Development Process

  1. First, I utilized my getallMessages and sendMessage function to send messages to the database and retrieve the messages later to use in the leadership board.
  2. Next, inside my gameOver() function, I used window.getAllMessages().then((value) to look through the array of scores given.
  3. Lastly, I wanted to select the top three scores in my array (the top scores) and arrange them in my leadership board. In order to do this I did "hiScores[0]" and so forth.

Issue Deep-Dive

  • I ran into many issues for this final prototype. My main issue was retrieving data from the Firebase database. How I fixed this issue is using the async function getAllMessages(). This function queries the database to get everything in the messages collection (which held my scores).
